# Lancer le projet 
###  Se mettre à la racine du projet : où se trouve le fichier docker-compose.yml
  Build le projet : docker-compose build  
  Lancer le projet : docker-compose up 

# Lancer MySQL

  docker exec -it projet-docker_db_1 bash  
  mysql -uroot -proot
